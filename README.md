# Online Bookstore Database Management and Analysis

## Overview
This project involves designing and analyzing a relational PostgreSQL database
for an online bookstore to manage books, customers, and orders efficiently.

## Tools & Technologies
- PostgreSQL
- SQL
- pgAdmin 4

## Database Design
- Normalized tables with primary and foreign key relationships
- Structured schema for books, customers, orders, and inventory

## Key Analysis Performed
- Total revenue analysis
- Most frequently ordered books
- Customer spending patterns
- Genre-wise sales insights
- Remaining stock after order fulfillment

## How to Use
1. Open pgAdmin
2. Create a new PostgreSQL database
3. Restore the `online_bookstore.sql` file
4. Run SQL queries to explore insights

## Key Learnings
- Relational database design
- SQL joins, aggregations, filtering, and grouping
- Practical PostgreSQL database management
